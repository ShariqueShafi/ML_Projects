# ML Projects

Welcome to my Machine Learning and Data Science Projects repository! This repository contains a collection of projects demonstrating my skills in data analysis, statistical modeling, machine learning, and text mining. 

Each project folder contains its problem statement, dataset, codebase (R/Python/.ipynb), and a dedicated README with the approach and key findings.

## Table of Contents

### Predictive Modeling & Classification
1. **[Medical Insurance Charge Prediction](./Projects/Insurance_Charge_Prediction)**
   - Predicting medical insurance charges using demographic and lifestyle factors with Multiple Linear Regression.
2. **[Housing Finance & Loan Analysis (Logistic Regression)](./Projects/Housing_Finance_Logistic_Regression)**
   - Analyzing applicant data to predict loan approval using Logistic Regression.
3. **[Housing Finance & Loan Analysis (Decision Tree)](./Projects/Housing_Finance_Decision_Tree)**
   - Predicting loan approval using Decision Trees for interpretable rule generation.
4. **[Total Cost to Hospital Analysis](./Projects/Hospital_Cost_Analysis)**
   - Identifying key drivers of total hospital costs incurred by patients during treatments using Regression.
5. **[Predicting LendingClub Loan Defaults](./Projects/MLWBA_LendingClub_Defaults)**
   - Building a classification model to predict loan defaults and reduce credit losses.

### Clustering & Recommendation
