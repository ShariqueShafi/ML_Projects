# Housing Finance & Loan Analysis

## Problem Statement
This project aims to analyze housing finance data to predict loan approval/amount requested using demographic and financial factors.

## Approach
Data preprocessing was done on the HousingFinance dataset. Logistic Regression and Decision Tree classification models were implemented to predict loan status or amount.

## Findings
Important factors such as credit history and applicant income play a major role in loan approval. Decision Trees provided interpretable rules for the approval process.
