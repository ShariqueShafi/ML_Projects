# YouTube Comments Text Data Modeling

## Problem Statement
To analyze YouTube comments and extract underlying conceptual frameworks and audience engagement metrics.

## Approach
Text data was scraped using the YouTube Data API. Regression models and hypothesis testing were used to evaluate the impact of different comment features on user engagement.

## Findings
Specific linguistic features and sentiments in comments significantly influence video engagement and viewer interaction.
