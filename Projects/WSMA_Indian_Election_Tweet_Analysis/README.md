# Indian Election 2019 Tweet Toxicity Analysis

## Problem Statement
To detect and analyze toxicity, hate speech, and aggression in tweets related to the 2019 Indian Elections.

## Approach
A dataset of election-related tweets was analyzed using NLP models to classify tweets based on toxicity levels.

## Findings
The analysis revealed patterns in aggressive political discourse and highlighted the prevalence of toxic speech during the election period.
