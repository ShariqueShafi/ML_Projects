# Housing Finance & Loan Analysis (Decision Tree)

## Problem Statement
This project aims to analyze housing finance data to predict loan approval using demographic and financial factors.

## Approach
A Decision Tree classification model was implemented to predict loan status, providing interpretable rules for the approval process.

## Findings
Important factors such as credit history and applicant income were identified as the root nodes for loan approval.
